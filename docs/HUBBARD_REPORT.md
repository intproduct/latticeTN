The AD loss path is unchanged; Stage 7C adds only the Hamiltonian/MPO/operator layer. Exact diagonalization is a reference baseline only.

## Known limitations

- 1D Jordan-Wigner fermions only; NOT a full graded fermionic tensor network. No TDVP, no finite-temperature, no graded tensors.
- Open-boundary spinful Hubbard chain only (d=4); no long-range hopping / interaction.
- Small systems only (N=4, chi=4/8); large-N scaling is out of scope.
- GPU is opt-in (LATTICETN_RUN_GPU=1) and selects only V100/TITAN V via the unified selector; clean-skips otherwise (no fallback).
- Runtime/speedup are recorded but the GPU is NOT required to be faster: small systems are overhead-dominated.
- ED is a CPU-only reference baseline; the GPU runs only the AD solver optimization.
- The dense reference hubbard_dense builds the global operators with the full 2N-mode Jordan-Wigner parity string; it is a genuine fermionic Hamiltonian, NOT a spin / hard-core-boson one.
